Rust Battlemetrics AFK Tool

The smart, automated solution for accumulating Battlemetrics playtime hours on Rust servers.

View on GitHub

Requires Python 3.x and the original Rust game client.

Core Features & Automation

Intelligent Server Rotation

Automatically switch between a predefined list of servers within a randomized time range (e.g., 2-3 hours). Mimics human behavior to avoid suspicion and keep playtime diverse across servers.

Stealth Mode & Anti-AFK

Features the optional "Kill after movement" functionality. This ensures your character is downed right after moving, preventing continuous spectating by admins and maintaining a low profile.

Full Console Automation

The tool handles the entire cycle: `client.disconnect`, `client.connect [IP]`, `respawn`, and simulated key presses (W + Space). No manual intervention is required after setup.

Adjustable AFK Interval

Easily customize the time interval between AFK cycles (1, 2, 5, or 10 minutes) directly through the GUI. Optimize the frequency of activity to your preference.

Integrated Server Manager

Manage your server list directly in the application. Add, remove, and categorize servers (Premium/Non-Premium) for easy selection and configuration of the rotation list.

Real-Time Status & Logging

Monitor the tool's activity with a real-time status display and a detailed log file that tracks every disconnect, connect, cycle count, and server switch for complete transparency.

Installation & Usage

Step 1: Prerequisites & Download

Ensure you have **Python 3.x** installed. Download the `rust_battlemetrics_hour_adder.py` file from the repository. The script will automatically install necessary dependencies (keyboard, pyautogui) upon its first run.

Step 2: Run the Tool

Execute the script from your terminal in the downloaded directory: 

python rust_battlemetrics_hour_adder.py

The tool will launch a simple, intuitive GUI for server management and control.
