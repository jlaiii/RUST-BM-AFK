Core Operational Features

  • BattleMetrics Hour Accumulation

    The core function. Utilizes a low-level Python automation engine to simulate human activity in Rust, ensuring your session is counted by BattleMetrics without you needing to be present.

  • Smart Server Rotation Logic

    Automatically rotates between a list of pre-selected servers on a timed interval (e.g., 1-2 hours) to maximize uptime and accrue hours across different environments, helping to avoid suspicion and mimic natural play patterns.

  • Minimal Activity & Stealth Modes

    Specialized modes to reduce detection risk. Minimal Activity Mode uses longer, 25-minute intervals, and Stealth Mode for rotation connects, kills the player instantly, and waits, ensuring minimal in-game presence.

  • Full Server Management GUI

    A user-friendly, tabbed graphical interface built with Tkinter for easy management, allowing users to add, remove, and filter server IPs (Premium/Non-Premium) for optimal farming.

  • Automated Game Management

    Includes features to automatically start Rust via Steam and focus the window upon boot-up. It also supports auto-restarting the game on a set schedule (e.g., every 6 hours) to handle potential game updates or crashes.

  • Windows Startup Integration

    Configure the tool to begin the hour farming process automatically upon Windows boot, ensuring seamless, 24/7 operation with minimal user intervention.
